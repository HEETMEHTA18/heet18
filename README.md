# Heet's Portfolio

A modern and responsive portfolio website built with React and Tailwind CSS.

## Features

- Responsive design for various devices
- Smooth scroll animations using Framer Motion
- Lazy loading of components for better performance
- Themed (Dark/Light mode - though currently defaults to dark)
- Sections for Hero, About Me, My Experience, My Portfolio (Projects), and Contact.

## Technologies Used

- React
- Tailwind CSS
- Framer Motion
- React Lazy Load / Suspense

## Setup and Installation

1. Clone the repository:

   ```bash
   git clone <repository_url>
   cd portfolio
   ```

2. Install dependencies:

   ```bash
   npm install
   # or
   # yarn install
   ```

## Running the Project

To run the project in development mode:

```bash
npm start
# or
# yarn start
```

The application will be available at `http://localhost:3000`.

## Building for Production

To build the project for production:

```bash
npm run build
# or
# yarn build
```

This will create a `build` folder with the production-ready files.
